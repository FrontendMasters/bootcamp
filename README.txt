How to fork a repository:
	1) Choose the repository which you want to fork in your profile.
	2) Go to that repository & at the top right corner, press Fork.
	3) Clone the repository on your local system using "git clone <link>". Here, link is the link for cloning which you can get from pressing Code.
	4) Now, enter the repository on your local system using change directory command.
	5) Create a new branch for the changes you'll make using "git checkout -b <new-branch-name>". After this the branch will change from master->new branch.
